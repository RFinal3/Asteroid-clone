# Asteroid Clone

A Pygame asteroid shooter that began as the Boot.dev Asteroids project and is being expanded into an original game.

## About

This is my first larger game project. I am using it to practice Python, object-oriented programming, game architecture, physics, collision detection, and reusable systems.

Rather than stopping at the tutorial requirements, I plan to develop it through several versions with new gameplay systems, levels, bosses, and roguelike progression.

## Current Features

- Momentum-based movement
- Screen wrapping
- Player lives and respawning
- Temporary respawn invulnerability
- Scoring and HUD
- Asteroid explosion effects
- Player speed limit and momentum decay
- Layered twinkling starfield
- Irregular asteroid shapes
- Triangular player hitbox
